Hola Mundo!

Aprendiendo a saludar en varios lenguajes de programación

Primero que nada, quiero darte la bienvenida a este blog, el cual es un proyecto personal que tenía en mente desde hace tiempo y ahora, con el pretexto de poner en práctica lo que he aprendido sobre diseño web y programación, por fin me animé a empezarlo.

Una de las primeras cosas que se aprende al estudiar cualquier idioma es a decir "Hola", y en los distintos lenguajes de programación que existen esa no es la excepción ya que ,en general, el primer programa que uno escribe es el que imprime la frase: "Hola Mundo!" en pantalla. Así que veamos cómo se saluda en algunos lenguajes de programación.

Saludando en el lenguaje C

El Lenguaje C, según los expertos, es uno de los más complejos que existen pero creo que toda persona interesada en la programación debería aprenderlo alguna vez, y aunque hay otros lenguajes como python que son más recomendados para empezar en el mundo de la programación, este lenguaje siempre debería estar en tu lista, junto con C++.
A continuación está el código en este lenguaje para saludar en pantalla.

#include <stdio.h> int main() { printf("Hola Mundo!"); return 0; }

Hi Python!

Python es actualmente uno de los lenguajes de programación más importante que existe y es muy amigable para las personas que quieran empezar a aprender a programar, además de que es muy versátil ya que se usa para el desarrollo web, la programación videojuegos, data science, etc.
Imprimir un saludo en pantalla en este lenguaje es tan sencillo como esto:

print('Hola Mundo!')

Ahora en Fortran

Para los profesionales de la ciencia, el lenguaje Fortran (Formula Translation) es una de las herramientas más poderosas que existen. Si te interesa saber un poco más sobre este lenguaje puedes checar esta página de documentación Fortran.
Para escribir "Hola mundo" en pantalla el código es el siguiente:

program HolaMundo write (*,*) "Hola mundo!" end program

Estos son algunos ejemplos de entre la vasta cantidad de lenguajes de programación que existen en la actualidad, y la ventaja es que en la web se pueden encontrar muchos recursos para aprender el que más se ajuste a tus necesidades. ¿Y tú, cuántos y cuáles lenguajes más conoces?
